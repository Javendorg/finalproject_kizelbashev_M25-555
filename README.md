## Инструкция по запуску программы:

1. Установка зависимостей: make install
2. Запуск программы: make project
3. Сборка проекта: make build
4. Публикация: make publish
5. Установка всех необходимых пакетов: make package-install
6. Проверка кода в соответствии с ruff: make line

## Инструкция по работе с приложением:

1.  **register** - регистрация пользователя (при регистрации выставляется стандартное значение счета: 10 000 USD)
2.  **login** - вход в аккаунт
3.  **portfolio** - просмотр финансового портфеля
4.  **buy** - покупка валюты 
5.  **sell** - продажа валюты
6.  **get-rate** - просмотр списка курсов валют
7.  **show-rates** - просмотр списка курсов валют (с фильтрацией)
8.  **update** - обновление списка валют 
9.  **parser** - состояние парсера
10. **autoupdate** - запуск автоматического обновления парсера
11. **stop** - остановка автоматического обновления парсера
12. **exit** - выход из приложения
